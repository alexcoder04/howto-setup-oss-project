 - Installation instructions
 - Usage
 - Configuration
 - Badges (https://shields.io)

 - specify Github topics
 - code of conduct
 - contributing guidelines

## Documentation

 - wiki
 - Github pages
 - man page

## Building / automation

 - Makefile
 - Dependabot / code quality checking

## Publishing

 - Github release
 - AUR
